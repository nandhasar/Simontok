✨ Fitur

Semua yang Anda Butuhkan

Fitur lengkap untuk monitoring task organisasi, dari input hingga export laporan.

📝

Input Task Cepat

Form input yang simpel dan intuitif. Tambah task dengan judul, status, prioritas, deadline, dan catatan dalam hitungan detik.

📊

Dashboard Interaktif

Visualisasi data lengkap dengan chart donut, bar chart, statistik real-time, dan kalender bulanan interaktif.

📅

Kalender Visual

Lihat semua task dalam tampilan kalender. Klik tanggal untuk melihat detail task yang jatuh tempo di hari tersebut.

🔍

Filter & Pencarian

Filter berdasarkan bulan, tahun, status, dan prioritas. Cari task secara real-time dengan search bar.

📄

Export PDF & Excel

Download laporan dalam format PDF atau Excel. Pilih rentang waktu atau export per bulan sesuai kebutuhan.

🌙

Dark & Light Mode

Tampilan nyaman untuk bekerja siang maupun malam. Tema tersimpan otomatis di browser Anda.
